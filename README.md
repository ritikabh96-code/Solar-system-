# Solar-system-
Solar system developed in unity is a basic visual structure, Creating the sun, planets and the moon using 3d spheres then later making them rotate, revolve and play sounds and allow the camera to follow the planets.
It is a solar system scene where the planets are placed around the sun revolving, the textures are applied on all the planets along with the sun, a star backhround is also added.
To add the rotation and revolution, create a c# script and use the same script for all the planets,then making all the planets rotate in their own axis then revolve around the sun, followed by making the moon revolve around the moon.
Then adding camera movements using another c# script.
